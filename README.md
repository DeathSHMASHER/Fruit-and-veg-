# Fruit-and-veg-
Two models run in parallel on every image:  YOLOv8 draws coloured bounding boxes around detected produce (apple, banana, orange, broccoli, carrot) HuggingFace ViT classifier covers 36 fruit/veg classes including mango, kiwi, eggplant, spinach, and more — runs on both the full image and each detected crop
